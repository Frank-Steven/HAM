# 0x04 HAM Δ 算符的分配律

Δ 算符 `<|` 对组合满足分配律（⊕ 表示任意二元算符）

```HAM
  (a <| b) ⊕ c
= a ⊕ c <| b ⊕ c
```

`c` 可以是任何表达式

```HAM
  (a <| b) ⊕ (c <| d)
= (a ⊕ (c <| d)) <| (b ⊕ (c <| d))
```

```HAM
  (a <| b) ⊕ x => x + 1
= (a ⊕ x => x + 1) <| (b ⊕ x => x + 1)
```

`<|` 对任何稳定的 Something 满足分配律

---

对于不稳定的 Something，则视为一个大函数。调用时，`<|` 两侧的函数分别接收参数，直到左侧稳定，用分配律变成稳定 Something。

---

Something 标准式

```HAM
... [函数] <| [函数] <| [值] <| [组合] <| [函数] <| [函数] ...
```

如果 `<| [值]` 在除了 `[组合]` 之外任何表达式后，则前面的所有表达式无效。

`[值] <| [组合]` 和 `[组合] <| [值]` 是等价表达，具体规则遵循覆写原则。

两个函数用 `<|` 连接时，参数同时传给两个函数，直到左侧稳定，用分配律变成稳定 Something。

```HAM
  (a <| b) <| c
= (a <| c) <| (b <| c)
```

上述等式成立。
